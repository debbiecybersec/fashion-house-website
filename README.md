# CSC106 Fashion House Website Project

## Project Title
Fashion House Website

## Course
CSC106 – Introduction to Web Technologies

## Description
This project is a simple fashion house website developed as part of the CSC106 course. The website showcases fashion products, provides information about the board of trustees, allows users to book appointments, and displays upcoming fashion events.

The website was created using basic web technologies such as HTML, CSS, and JavaScript and is hosted using GitHub.

## Features
- Home page introducing the fashion house
- Product showcase page displaying fashion items
- Board of Trustees page showing the team members
- Appointment booking page for customer inquiries
- Upcoming events page displaying fashion shows and promotions
- Consistent navigation links across all pages
- Scrolling marquee displaying announcements
- External CSS file for styling
- JavaScript for interactive functionality
